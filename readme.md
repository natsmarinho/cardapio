## Como rodar a aplicação :arrow_forward:

No terminal, clone o projeto: 

```
git clone https://github.com/natsmarinho/cardapio.git
```
Após clonar, coloque os comandos ainda dentro do terminal: 
``` 
npm run build
```
```
npm install
```
```
run dev
```
